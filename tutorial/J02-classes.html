Классы в JavaScript.

JavaScript поддерживает программирование в объектно-ориентированном стиле.

Классы.

Для описания класса используется ключесое слово class, за которым записывается имя класса (с заглавной буквы), а само описание класса заключается в фигурные скобки. Пример описания класса "Человек": 
# Описание класса
class Person {
  // конструктор объекта
  constructor(gender, birthday, name) {
    this._gender = gender;
    this._birthday = birthday;
    this._name = name || "";
    this._deathday = null;
    this._spouse = null;
    this._children = [];
  }

  // статические свойства класса = константы
  static get FEMALE() { return 0; }
  static get MALE() { return 1; }
}
# Создание объекта (экземпляра класса):
leo = new Person(Person.MALE, "1988-6-24", "Lev Shock"));
