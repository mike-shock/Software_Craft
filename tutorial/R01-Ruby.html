Ruby.

Ruby — интерпретируемый динамический полностью объектно-ориентированный  высокоуровневый язык программирования. Разработан под влиянием языков Perl, Smalltalk, Eiffel, Ada и Lisp, а также C++, CLU, Dylan, Lua и Python.

Ruby поддерживает несколько парадигм программирования: процедурную (определение подпрограмм и переменных вне классов), объектно-ориентированную (всё является объектами), функциональную (анонимные функции, замыкания, возврат значения всеми инструкциями, возврат методом последнего вычисленного значения, функции высшего порядка), аспектно-ориентированную (AOP).

В нём есть средства всесторонней интроспекции (reflection), реализована независимая от ОС многопоточность и эффективный сборщик мусора, имеются мощные средства мета-программирования.
История создания.
Разработал язык Ruby японский программист Юкихиро Мацумото, известный под ник-неймом Matz. Название языка навеяно языком Perl, многие особенности синтаксиса и семантики из которого заимствованы в Ruby.

Одним из источников вдохновения для Ю. Мацумото при разработке Ruby был научно-фантастический роман «Вавилон-17»  Сэмюэля Дилэни, основанный на гипотезе лингвистической относительности Сепира – Уорфа (язык определяет мышление, и, соответственно, лингвистические категории ограничивают и определяют когнитивные категории). Поэтому синтаксис и средства языка программирования во многом определяют способ профессионального мышления разработчика: его подход к задаче и манеру программирования.

Целью разработки было создание настоящего объектно-ориентированного интерпретируемого мощного и удобного языка программирования, лёгкого в изучении, который приятного использовать, на котором можно быстро разрабатывать программы.

Matz создал сбалансированный гармоничный язык с ясным предсказуемым синтаксисом, сочетающий преимущества объектно-ориентированного программирования с достоинствами других парадигм.

Ruby реализован для всех современных аппаратных платформ и операционных систем. Реализации языка Ruby — свободно распространяемые (под лицензиями Ruby License, GPLv2, 2-clause BSD License).

Примеры: 
// Ruby:
  def print_message(string, n) // описание подпрограммы начинается с 'def'
    puts string + "!"*n        // операция '*' повторяет строку n раз
  end                          // и заканчивается 'end'
  message = "Миру - мир"       // ';' в конце строки можно не ставить
  print_message message, 3     // вызов подпрограммы print_message(message, 3) можно писать без скобок
                               // --> Миру - мир!!!
Установка.

Скачать свежий дистрибутив системы программирования Ruby можно с официального сайта языка: ruby-lang.org/ru/downloads/.

GNU/Linux.

Обычно в ОС Linux Ruby уже установлен. Но если нет, то его можно установить с помощью cистемы управления пакетами, например:
    sudo apt-get install ruby # дистрибутивы на основе Debian
    sudo yum install ruby     # дистрибутивы на основе RedHat
Как установить Ruby в других дистрибутивах, описано в документации, в том числе на русском языке.

MacOS.

На MacOS, скорее всего, Ruby уже установлен. Это можно легко проверить: откройте терминальное окно и введите команду:
ruby -v

В ответ вы должны увидеть версию установленного интерпретатора языка.
Если программа-интерпретатор не найдена, то для установки можно использовать сторонние инструменты: rbenv и RVM.

Microsoft Windows.

На официальном сайте языка Ruby для MS Windows предлагается использовать установщик с сайта rubyinstaller.org Надо скачать оттуда установщик и запустить его.
По умолчанию Ruby ставится в /Program Files/, а я обычно устанавливаю в каталог /opt/ruby/.

Ruby gems.

Как и для других языков программирования, для Ruby разработана удобная система установки дополнительных библиотечных модулей, которые называются gems. За управление такими модулями отвечает утилита gem.
Например, для запуска тестового web-сервера puma + rack потребуется установить такие "самоцветы":
gem install http_router
gem install rack
gem install puma
Посмотреть список установленных модулей можно командой:
gem list
Обновить модуль можно командой: 
gem update rack
Удалияется модуль командой: 
gem uninstall puma
Остальные команды для управления модулями выводятся командой: 
gem help commands

Q&A.
Q: Как же всё-таки зовут разработчика языка Ruby?
A: Matsumoto Yukihiro (松本 行弘). Мацумото (まつもと) — фамилия, Юкихиро (ゆきひろ) — имя, а Matz — прозвище.

Q: Что самое главное в языке Ruby?
A: Гармония. Соразмеренное сочетание выразительных синтакисческих конструкций с мощью и логичностью языковых средств.

Q: Почему язык Ruby хорошо подходит для обучения, особенно в качестве первого языка?
A: Ruby — чисто объектный непротиворечивый язык, предсказуемо и удобно построенный, прост для освоения и очень хорошо годится для понимания объектно-ориентированного программирования.
