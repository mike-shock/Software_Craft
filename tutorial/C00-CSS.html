CSS.

Стили.

Каскадные стилевые таблицы (Cascading Style Sheets) дополняют HTML. С помощью CSS мы описываем, как должно быть представлено содержимое документа HTML.
Чаще всего стилевые таблицы хранятся в отдельных файлах с суффиксом (расширением) .CSS и подключаются в тех HTML-файлах, к оторым нужно применить стили.

Примеры: 
# CSS встроенная и внешняя:
  <html>
    <head>
      <!-- встроенная стилевая таблица -- >
      <style>
        h1 {
          font-weight: bold;
          font-size: 8mm;
        }
      </style>
      <!-- ссылка на внешнюю стилевую таблицу -- >
      <link rel="stylesheet" type="text/css" href="document.css" />
    </head>
    <body>
      <h1> Этот заголовок будет выведен полужирным шрифтом буквами размером 8 мм </h1>
    </body>
  </html>
Из чего состоят описания стилей.

Q&A.
Q: Можно программировать на CSS?
A: CSS — это язык описания стилей, а не язык программирования. Но на CSS можно описать преобразования (transformation), а браузер по описанию преобразований будет динамически изменять объекты на странице. Таким образом можно описать несложную анимацию. Например:
  <!-- В CSS описываем transformation с именем emergeFromEmptiness -->
  <style>

    @keyframes emergeFromEmptiness {
       0%   { opacity: 0; }
       100% { opacity: 1; }
    }
    #santa {
       animation-name: emergeFromEmptiness;
       animation-duration: 5s;
       animation-delay: 0s;
    }

  </style>
  <!-- В HTML назначаем id "santa" для элемента, к которому будет применяться transformation -->
  <img id="santa" src="images/FatherFrost.png" height="500" />
