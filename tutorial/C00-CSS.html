CSS.

Стили.

Каскадные стилевые таблицы (Cascading Style Sheets) дополняют HTML. В них указываются описания, как должно быть представлено содержимое документа HTML.
Чаще всего стилевые таблицы хранятся в отдельных файлах с суффиксом (расширением) .CSS и подключаются в тех HTML-файлах, к оторым нужно применить стили.

Примеры: 
# CSS встроенная и внешняя:
  <html>
    <head>
      <!-- встроенная стилевая таблица -- >
      <style>
        h1 {
          font-weight: bold;
          font-size: 8mm;
        }
      </style>
      <!-- ссылка на внешнюю стилевую таблицу -- >
      <link rel="stylesheet" type="text/css" href="document.css" />
    </head>
    <body>
      <h1> Этот заголовок будет выведен полужирным шрифтом буквами размером 8 мм </h1>
    </body>
  </html>
Q&A.
Q: Можно программировать на CSS?
A: Нет, это язык описания стилей, а не язык программирования. Но на CSS можно описать преобразования (transformation), которые заставят браузер динамически изменять .
