Содержание
JavaScript.

JavaScript (JS) — язык, созданный Бренданом Эйхом (из компании Netscape), который поддерживает императивный, функциональный и объектно-ориентированный (на основе прототипов) стили программирования. Это реализацией спецификации ECMAScript (стандарт ECMA-262). JavaScript наиболее широко применяется в браузерах как язык сценариев для придания интерактивности веб-страницам.

JS в HTML-документе.

Для добавления кода на JavaScript в HTML можно использовать теги <script> и </script>. Контейнеров <script> в одном документе может быть сколько угодно. Атрибут type="text/javascript" указывать в теге <script> необязательно, это значение используется по умолчанию.

Пример скрипта, выводящий модальное окно с текстовую строку: 

# Язык программирования: JavaScript
<script type="application/javascript">
alert("Миру - мир!");
</script>

Программы на JavaScript рекомендуется выносить в отдельные файлы (с суффиксом .js) и подключать их с помощью тега <script>:

<script type="application/javascript" src="http://путь/к/файлу/со/скриптом.js"> </script>

Подключения внешних скриптов принято помещать внутри контейнера <head></head>.


JS в атрибутах событий.

У тегов HTML могут быть атрибуты, в которых указываются действия (на языке JavaScript) для обработки определённых событий, например:

<button onclick="alert('Миру - мир!');">Сказать</button>

Синтаксис.

Переменные.

Переменные для хранения данных можно объявлять несколькими способами:

// объявить переменную `email` с неопределённым начальным значением `undefined`
var email;

// объявить переменную и явно присвоить ей неопределённое значение
var address = undefined;

// объявить для блока переменную `counter` (со значением `undefined`)
let counter;

// объявить  для блока неизменяемую переменную `z` с начальным значением
const slogan = "Это значение нельзя переопределить!";

// объявить переменную `telephone` и присвоить ей числовое значение
let telephone = 911;

// заменить значение переменной `telephone` строковым значением
// JavaScript is a dynamically-typed language, so this is legal.
telephone = "911";

Вывод данных

Для проверки, отладки и протоколирования можно выводить данные в консоль браузера:

console.log("Миру - мир!");

Можно выводить данные в документ HTML (внутри тега <script>):

// добавить в DOM текстовый объект
document.write("Миру - мир!");

Подпрограммы.

В JavaScript все подпрограммы — это функции, которые могут возвращать результат командой return, если это нужно. 

// описание функции
function hello_user(user_name) {
  alert("Привет, "+user_name+"!");
}

// вызов функции = обращение к функции
hello_user("Брендан Эйх"); // --> "Привет, Брендан Эйх!"
Рекурсивная функция вызывает сама себя: 
function factorial(n) {
  if (n === 0) {
    return 1; // 0! = 1
  } else {
    return n * factorial(n — 1);
  }
}
factorial(3); // --> 6

Анонимная функция — это описание функция без имени, но при этом создаётся программный объект «функция», который можно присвоить в переменную и вызывать по имени этой переменной. 

//
var hello_user = function(user_name) { alert("Привет, "+user_name+"!"); }

// вызов анонимной функции выглядит как вызов обычной функции
hello_user("Брендан Эйх"); // --> "Привет, Брендан Эйх!"

Можно вызвать анонимную функцию (но только один раз), не присваивая её переменной. 

// описание и немедленный вызов анонимной функции
(function(user_name){ alert("Привет, "+user_name+"!"); })("Брендан Эйх");

Анонимную функцию удобно передавать в подпрограмму для выполнения действий, например, по таймеру: 

// установить интервал в 1000 миллисекунд для автоматического вызова функции
setTimeout(function(){ /* команды */ }, 1000);

или указывать в качестве обработчика события:

window.onload = function() { alert('Страница успешно загружена.'); }

Альтернативы.

TypeScript

TypeScript — язык программирования, разработанный в 2012 году Андерсом Хейлсбергом (из Microsoft), автором языков Turbo Pascal, Delphi и C#. TypeScript позиционируется как средство разработки веб-приложений, расширяющее возможности JavaScript. В TypeScript, в отличие от JavaScript, есть возможность явного статического назначения типов, поддержка использования полноценных классов (как в традиционных объектно-ориентированных языках), поддержка подключения модулей: всё это призвано повысить скорость разработки, облегчить читаемость, рефакторинг и повторное использование кода, помочь осуществлять поиск ошибок на этапе разработки и компиляции, и, возможно, ускорить выполнение программ.
