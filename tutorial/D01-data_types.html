Типы данных.

Обучение программированию начинается с примеров, в которых используются разные типы данных (data type) — целые числа (integer), числа с плавающей точкой (float, double), текстовые строки (string), логические величины (boolean, Bool). В большинстве языков программирования, как компилируемых (Java, Kotlin, C#, C++), так и интерпретируемых, «скриптовых» (Perl, Ruby, JavaScript, Python, PHP), такие данные обычно соответствуют машинным словам, последовательностям байтов и битам в оперативной памяти.
В любом языке программирования есть простые или скалярные (scalar) типы данных (integer, float, string, boolean, ...). В скалярной переменной (variable) можно хранить только одно значение (value) определённого типа, а затем заменять его операцией присваивания (assignment): 
# Ruby, Python:
    email = "deinlandel@gmail.com"
    age = 32
# JavaScript:
    var profession = "программист";
    let alive = false;
При этом часто при присваивании значения справа от имени переменной после знака присваивания (=) записываются литералы (literals), то есть непосредственные значения, константы (literals): 
    " роман 'Война и мир' "     # строка в кавычках, string
    ' роман "Война и мир" '     # строка в апострофах, string
    -42                         # отрицательное целое число, integer
    3.14                        # дробное число, float
    1Е+6                        # дробное число =1*106, float
    true                        # логическое значение, boolean
Кроме присваивания для каждого типа данных в языке предусмотрен свой особенный набор операций (operation):
  • для числовых (numeric) типов данных — арифметические (сложение, деление, ...) и другие математические,
  • для булевых (boolean) — логические (И, ИЛИ, НЕ, ...),
  • для строк (string) — строковые (сцепление, выделение подстроки, нахождение её длины, ...).
Часто встречающиеся операции обозначаются традиционными (+, -, /, *, >, <, >=, ...) или специальными (++, ==, %, !=, +=, |, &, ^, <<, ||, ...) символами, а также ключевыми словами языка (and, or, not, ...). Другие операции существуют в языке в виде библиотечных функций (abs, sin, substr, length, slice, ...). Обычно в языке программирования есть особое скалярное значение для обозначения неопределённого значения (undefined value) — указания на то, что переменной не присвоено никакого значения: nil (Ruby, Pascal, Clojure), undef (Perl), null (Java, Kotlin, JavaScript, PHP), NULL (C++, SQL), None (Python), Nothing (Scala). Примеры операций над скалярными данными: 
    2 * PI * radius
    temperature > 0
    "текстовые строки " + "можно сцеплять" + "в одну"
    true or false
Строки, содержащие текстовую информацию, в некоторых языках рассматриваются не как скалярные данные, а как последовательности символов или байтов: по сути, массивы (списки) символов. Примеры операций над скалярными строками: 
# Ruby
first = (string.nil? || string.length()==0) ? "" : string.slice(0,1)
# Python
first = '' if (string == None or len(string)==0) else string[0]
В современных языках программирования текстовые строки хранятся в многобайтовой кодировке (чаще всего в UTF-8), где один символ может представляться несколькими байтами. При разработке большинства программ возникает необходимость использовать более сложные структуры данных.
