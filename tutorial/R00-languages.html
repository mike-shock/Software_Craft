Языки программирования.

От выбора языка программирования зависит многое: потому, что средства языка дают инструменты для решения задачи и влияют на мышление программиста. Некоторые языки облегчают решение конкретной задачи, поскольку в них есть удобные средства для этого. Поэтому решаемая задача может повлиять на выбор языка. Естественно, разработчику проще писать на хорошо знакомом языке.

Не стоит выбирать язык, если его особенности мешают программисту:
  • сложный синтаксис, много специальных обозначений (Perl);
  • язык неудачно спроектирован (PHP);
  • неоправданное усложнение синтаксиса (например, Python навязывает зависимость от пробелов);
  • ООП только имитируется (Python, Perl);
  • несовместимость при смене версий (PHP, Python);
Лучше выбирать язык программирования, который помогает программисту своими выразительными средствами:
  • язык спроектирован сбалансированно;
  • язык нацелен на удобство для разработчика;
  • синтаксис языка простой и понятный;
  • конструкции языка логичные и предсказуемые;
  • средства языка мощные и выразительные:
  • возможность не писать необязательные конструкции улучшает читабельность;
  • программы получаются короткие и ясные;
Поэтому из скриптовых языков наиболее привлекательный — Ruby, а среди компилируемых — Kotlin превосходит язык Java.
Q&A.
Q: Какие языки программирования нужно знать разработчику?
A: Разработчику часто приходится знать несколько языков:
  • современный (компилируемый) объектно-ориентированный язык (object-oriented language): Java / Kotlin / C#, C++, ...
  • динамический ("скриптовый") язык (dynamic language): Ruby, Python, ...
  • встроенный в браузер язык: JavaScript / TypeScript
  • командный язык операционной системы: sh, cmd / PowerShell
  • язык запросов к реляционным базам данных: SQL
Желательно познакоммиться с одним из языков функционального программирования (functional programming): Scala, Erlang, Haskell, ...
