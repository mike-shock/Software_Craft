Программа. 
  Любую программу можно разделить на нескольких частей, которые отличаются по назначению синтаксических конструкций.
  
  Обычно в начале программы подключаются программные библиотеки (пакеты), в которых содержатся нужные подпрограммы.
  
  // JavaScript:
  <script src='library.js'></script>

  # Ruby:
  require 'date'

  # Python:
  import datetime
  
  Затем размещаются описания подпрограмм, которые будут вызываться в основной программе.
  
  // JavaScript:
  function weekday(weekday_names) {
    var now = new Date();
    var day = now.getDay();
    var weekday = weekday_names[day];
    return weekday;
  }

  # Ruby:
  def weekday(weekday_names)
    now = Time.now
    var day = now.wday
    var weekday = weekday_names[day]
    return weekday
  end
  
  Далее размещаются описания переменных для данных, которые будут использоваться в основной программе.
  
  // JavaScript:
  var weekday_names = ['воскресенье', 'понедельник', 'вторник', 'среда', 'четверг', 'пятница', 'суббота'];

  # Ruby:
  weekday_names = ['воскресенье', 'понедельник', 'вторник', 'среда', 'четверг', 'пятница', 'суббота']
  
  В конце следуют команды обработки данных в основной программе. Среди них обязательно будут команды ввода-вывода.
  
  // JavaScript:
  <script>
    var today = weekday(weekday_names);
    document.write(today);
  </script>

  # Ruby:
    today = weekday(weekday_names)
    puts today
  
  

  Q&A.
  Q: Какие есть правила для записи имён переменных?
  A: Почти во всех языках переменные должны записываться латинскими буквами и цифрами, 
  причём первая должна быть буква. в имени переменной допускается знак подчёркивания.
  Переменные и имена подпрограмм чаще всего записываются либо так:
  
  this_is_a_variable_name
  this_is_a_function_name()
  
  либо так (camel case):
  
  thisIsAVariableName
  thisIsAFunctionName()
  
  Константы часто записываются заглавными буквами:
  
  PI = 3.141592653
